# Celebal Technologies Signup Form

A modern, responsive signup form built with React, TypeScript, and Tailwind CSS.

##  Form Fields
- First Name & Last Name
- Email Address
- Password (with strength requirements)
- Confirm Password
- Phone Number

## Technologies Used

- **React** 
- **TypeScript** 
- **Vite**
- **Tailwind CSS**
